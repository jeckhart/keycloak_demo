# Vagrant Setup

### Setup
* Install [vagrant]
 * Install [vagrant-vbguest] plugin
* Install [VirtualBox]
* Install Ansible: http://docs.ansible.com/intro_installation.html
* install ansible-galaxy roles
    ansible-galaxy install -r requirements.yml


### Running
* `vagrant up`
